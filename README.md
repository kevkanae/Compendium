## Compendium

A new Flutter project, that is a search engine for medicinal drugs that displays data directly from [FDA](https://open.fda.gov/)

## Project Milestones

| Milestones                     | Progress |
| ------------------------------ | -------- |
| Firebase/Firestore Integration | ✔        |
| Login/Sign-Up                  | ✔        |
| 2 Seperate API Calls           | ✔        |
| Data Display                   | ✔        |
| JSON Decode                    | ✔        |
| Use Firestore as DB for Info   | ❌       |
| Code Refactors                 | ❌       |
| UI remake                      | ❌       |
